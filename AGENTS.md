## Agent skills

### Issue tracker
Issues are tracked in GitHub Issues for `nicklausroach/arbor`; external PRs are not a request triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels
Uses the default engineering triage labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs
Single-context repo. Read root `CONTEXT.md` and `docs/adr/` when present. See `docs/agents/domain.md`.
